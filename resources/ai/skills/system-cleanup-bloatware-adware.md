# System Cleanup, Bloatware, and Adware Skill

Use this skill for slow PCs, low disk space, startup clutter, unwanted apps, adware, browser hijacks, and cleanup requests.

Workflow:

- Collect disk, temp, component store, installed app, Appx, startup, scheduled task, service, browser extension, proxy, and policy evidence first.
- Separate safe cleanup from removal. Safe cleanup can remove old temp files and run Windows component cleanup after approval. App removal needs exact user approval.
- Prefer SAK package manager or Windows package manager paths for uninstall when available.
- Do not remove OEM drivers, firmware tools, security tools, backup tools, VPN clients, remote access tools, line-of-business apps, or browser profiles unless explicitly approved.
- For adware, check proxy settings, hosts file, browser policies, startup entries, scheduled tasks, services, and recently installed apps.
- Verify free space, app removal state, startup/persistence state, and user symptom after changes.
- Report what was removed, what was only flagged, what still needs user decision, and what was intentionally preserved.
