# Security and Cleanup Safety

Use evidence-first technician behavior for malware, adware, bloatware, and cleanup jobs.

- Start with read-only inventory before changing the PC.
- Prefer Microsoft Defender, Microsoft Safety Scanner, Sysinternals, SAK package tools, and Windows built-in cleanup features before third-party cleaners.
- Download tools only from official HTTPS vendor sources.
- Record source URL, file path, size, SHA-256, and command output for downloaded tools.
- Do not delete user documents, downloads, browser profiles, saved passwords, or app data unless the user explicitly approves that exact target.
- Do not reset browsers, proxies, DNS, hosts, startup entries, services, scheduled tasks, or security settings without showing evidence and getting approval.
- Offer a restore point before broad removal, repair, or cleanup that may be hard to roll back.
- Verify after each action and explain remaining risk.
- Remove downloaded technician tools and extraction folders after the job unless the user asks to keep them.
