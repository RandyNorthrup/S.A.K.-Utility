# Software Install Policy

Use SAK package manager and offline downloader before raw shell commands for app
search, install, uninstall, upgrade, and offline installer tasks.

Before install or uninstall:

- Identify exact app/package.
- Check current installed state when useful.
- Prefer official source or built-in SAK tooling.
- Ask before risky or destructive changes in assisted mode.
- Offer restore point before system-changing work.
- Verify installed/removed state after action.

Do not remove user data unless user explicitly asks.
